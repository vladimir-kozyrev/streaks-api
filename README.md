# STREAKS API

An impression of what [STREAKS](https://streaksapp.com/) API could look like.
This is a practice project that will help me learn FastAPI.

## Installation

```shell
$ pipenv install
$ pipenv shell
```

## Run it

```shell
$ uvicorn main:app --reload
```
